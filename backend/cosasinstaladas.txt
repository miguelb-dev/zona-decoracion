GUIA DE COMANDOS - PROYECTO ZONA DECORACION

1. GESTION DEL ENTORNO VIRTUAL (PYTHON)

Crear el entorno virtual (se hace una sola vez al iniciar el proyecto):
python -m venv venv

Activar el entorno virtual en Windows PowerShell:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned ;
.\venv\Scripts\Activate.ps1

(Sabrás que funcionó porque aparecerá la palabra (venv) al inicio de la línea en tu terminal).

2. INSTALACION DE LIBRERIAS DEL BACKEND (PYTHON)

Instalar herramientas de Base de Datos y lectura de archivo .env:
pip install sqlalchemy pymysql python-dotenv

Instalar herramientas del Servidor API y validaciones de datos:
pip install fastapi pydantic uvicorn

* NOTA DE NOTIFICACIONES: El sistema de envío de correos utiliza 'smtplib' (nativo de Python) y 'BackgroundTasks' (nativo de FastAPI), por lo que NO requiere ninguna instalación extra vía pip.

Exportar la lista de librerías instaladas a un archivo de texto (opcional para compartir):
pip freeze > requirements.txt

3. COMANDOS DE VERIFICACION Y EJECUCION DEL BACKEND

Verificar la conexión limpia a MySQL (XAMPP):
python backend/database/conexion.py

Verificar que el modelo de pintura mapee bien con la base de datos:
python -m backend.modules.pintura.models

Encender el servidor backend de FastAPI con auto-recarga automática:
uvicorn backend.main:app --reload

(Para apagar el servidor en la terminal, presiona la combinación de teclas CTRL + C).

4. COMANDOS DEL FRONTEND (NODE.JS / VITE / REACT)

Instalar los módulos de JavaScript (se hace una sola vez para generar la carpeta node_modules):
npm install

Encender el servidor de desarrollo del Frontend (Vite):
npm run dev
